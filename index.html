Formula Help

Mathematical Functions

  • =SUM(range) - Calculates the sum of a range of cells. Example: =SUM(A1:A5)
  • =AVERAGE(range) - Calculates the average of a range of cells. Example: =AVERAGE(B1:B10)
  • =MAX(range) - Returns the maximum value from a range of cells. Example: =MAX(C1:C20)
  • =MIN(range) - Returns the minimum value from a range of cells. Example: =MIN(D1:D15)
  • =COUNT(range) - Counts the number of cells containing numerical values in a range. Example: =COUNT(E1:E30)

Data Quality Functions

  • =TRIM(cell) - Removes leading and trailing whitespace from a cell. Example: =TRIM(A1)
  • =UPPER(cell) - Converts the text in a cell to uppercase. Example: =UPPER(B5)
  • =LOWER(cell) - Converts the text in a cell to lowercase. Example: =LOWER(C10)
  • =FIND_AND_REPLACE(range, find, replace) - Finds and replaces text in a range. Example: =FIND_AND_REPLACE(A1:A10, "old", "new")
  • =REMOVE_DUPLICATES(range) - Removes duplicate values from a range. Example: =REMOVE_DUPLICATES(A1:A10)
